**Requisitos previos:**
-   Node.js               
-   npm o yarn

**Pasos:**

1. Clona el repositorio:
    ```bash
    git clone https://github.com/Titin02/pokeAPI.git
    ```
2. Navega al directorio:
    ```bash
    cd pokeAPI/
    ```
3. Instalar las dependencias:
   ```bash
   npm install
   ```
4. Iniciar el servidor:
   ```bash
   npm run dev
   ```
